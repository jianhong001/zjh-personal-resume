# 赵健宏 · AI 产品经理方向个人网站

一个轻量静态个人简历与案例作品集网站，服务于 AI 产品经理方向求职。

## 内容定位

- AI 视频生成质量评估与 Benchmark 治理
- AIGC 内容生产工作流与图生视频一致性控制
- AI 产品运营、客户成功与产品体验反馈
- 开发者社区增长与数据监控

## 项目结构

```
personal-resume/
├── index.html
└── assets/
    ├── css/main.css
    ├── js/main.js
    └── images/
        └── cases/
```

## 使用方式

直接在浏览器打开 `index.html` 即可预览，不依赖构建工具。

公开信息默认只展示姓名与邮箱，不展示手机号；涉及腾讯云雀实习的内容只保留职责、方法与通用经验，不暴露内部敏感细节。
